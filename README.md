# Billig-Reiser.no

Ferdig Vite/React landingsside med affiliate-lenker, hero-bilde, logo og klargjort `/api/flights.js` for Amadeus senere.

## Kjør lokalt

```bash
npm install
npm run dev
```

## Deploy til Vercel

1. Last opp repoet til GitHub.
2. Importer repoet i Vercel.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
