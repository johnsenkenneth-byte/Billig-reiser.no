# Billig-reiser.no

Wireframe-match versjon med mørk deal-radar, interaktiv logo, mock deals og Vercel API-rute klar for Amadeus.

## Kjør lokalt
```bash
npm install
npm run dev
```

## Deploy til Vercel
1. Last opp til GitHub
2. Importer repo i Vercel
3. Deploy
4. Legg Amadeus-nøkler som miljøvariabler senere

## API
`/api/flights.js` er en placeholder som kan byttes ut med ekte Amadeus-kall.
